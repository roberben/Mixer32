Instalador Pro MIXER32

Prepara tu hardware MIXER32 para la acción instalando su firmware base.

1. Conecta tu dispositivo al ordenador por USB.

2. Haz clic en "Connect", selecciona el puerto UART y confirma la instalación.

Recursos del Proyecto

💻
Aplicación Linker32 (Releases) El hardware Mixer32 requiere la app de escritorio Linker32 para funcionar. Descarga aquí la última versión para Windows.
📖
Repositorio de GitHub Consulta la documentación completa, el código fuente del proyecto y comprende cómo está estructurado por dentro.
🖨
Carcasa 3D en MakerWorld Descarga el modelo físico, prepáralo en tu laminador 3D favorito e imprime la carcasa oficial para montar tu Mixer32.
⚠
Importante:

Esta herramienta utiliza la Web Serial API, soportada por navegadores como Google Chrome o Microsoft Edge en escritorio.

Asegúrate de que no haya otras aplicaciones accediendo al puerto serie del ESP32 antes de intentar conectar.
