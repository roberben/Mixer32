Instalador Web MIXER32

Conecta tu dispositivo ESP32 (MIXER32) al ordenador mediante un cable USB.

Haz clic en el botón "Connect" a continuación, selecciona el puerto serie correspondiente (por ejemplo, USB a UART) e instala la última versión del firmware.

Nota: Esta herramienta requiere un navegador que soporte la Web Serial API (como Google Chrome o Microsoft Edge en escritorio).
